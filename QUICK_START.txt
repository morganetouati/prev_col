🚀 COMPILATION EN 2 ÉTAPES:

   ⭐ SOLUTION 1: Android Studio (RECOMMENDED)
   ═════════════════════════════════════════════
   1. Installez Android Studio: https://developer.android.com/studio
   2. File > Open > C:\Users\morgane\Desktop\prev_col
   3. Attendez la synchro Gradle (5-10 min)
   4. Build > Build Bundle(s)/APK(s) > Build APK(s)
   5. APK généré automatiquement → app/build/outputs/apk/debug/

   ✓ Gère tout automatiquement (JDK, Gradle, SDK)
   ✓ Meilleur support pour Windows
   ✓ Peut tester directement sur un appareil

   ───────────────────────────────────────────

   ⭐ SOLUTION 2: Gradle local (avancé)
   ═════════════════════════════════════════════
   1. Installez Gradle 7.6+: https://gradle.org/releases/
   2. Ajoutez Gradle au PATH (variables d'environnement)
   3. Ouvrez CMD dans C:\Users\morgane\Desktop\prev_col
   4. Tapez: gradle assembleDebug
   5. APK généré dans app/build/outputs/apk/debug/

🎯 RÉSULTAT: l'APK sera dans:
   app/build/outputs/apk/debug/app-debug.apk
